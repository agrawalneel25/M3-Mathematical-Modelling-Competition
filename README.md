# M3 Mathematical Modelling Competition

Submission for the MathWorks Math Modeling Challenge.

This project models homelessness trends using time-series forecasting and statistical relationship testing. The work was completed as part of a team submission shortlisted from a global competition field.

## What Is Included

- Final submitted modelling paper: [`M3 Submission (3).pdf`](M3%20Submission%20%283%29.pdf)
- Forecasting approach using ARIMA-style time-series modelling.
- Predictor analysis using Granger causality to test whether candidate variables help explain future homelessness trends.
- Policy-facing interpretation of model outputs rather than a purely technical report.

## Technical Focus

- Time-series preprocessing and forecasting.
- Model selection under competition time constraints.
- Statistical testing for lead-lag relationships.
- Communicating quantitative uncertainty to a non-technical audience.

## Notes

The repository currently contains the final paper rather than a full reproducible codebase. The paper documents the modelling assumptions, methodology, results, and recommendations.
